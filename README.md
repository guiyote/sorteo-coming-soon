# sorteo.elrevoltijo.com.uy

Página estática de cierre del sorteo de los 30 televisores de [El Revoltijo](https://www.elrevoltijo.com.uy).

## ¿Qué es?

Una landing page minimalista alojada en `sorteo.elrevoltijo.com.uy` que informa a los usuarios que el sorteo finalizó y los redirige al sitio principal.

## Archivos

| Archivo | Descripción |
|---|---|
| `index.html` | Página principal (HTML + CSS inline, sin dependencias) |
| `logo.png` | Logo de El Revoltijo |
| `CNAME` | Apunta el dominio `sorteo.elrevoltijo.com.uy` a GitHub Pages |
| `usuarios-*.csv` | Exportación de participantes del sorteo |

## Despliegue

El sitio se sirve directamente desde GitHub Pages. No requiere build ni dependencias — es HTML puro.
